Remarks
This is used to allow the AttributeSetContainer extension to act as mutators for the otherwise immutable AttributeSet. Each method of the sextension class will read the attribute set from the container and create a new set based on the paramaters (addding or removing attributes fom the set) producing a new attributeSet that is then re-assigned back to the container.
